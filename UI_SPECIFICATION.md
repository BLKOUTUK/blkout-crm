## 1. Design Principles

### Values-Aligned Design
- **Liberation**: Empowering users with full data visibility and control
- **Community**: Warm, inclusive visual language
- **Democratic**: Equal access, transparent processes

## 4. Component Library

### Buttons

```tsx
// Primary
<Button variant="default">Save Changes</Button>

// Secondary
<Button variant="outline">Cancel</Button>

// Destructive
<Button variant="destructive">Delete</Button>

// Ghost
<Button variant="ghost">View Details</Button>

// With Icon
<Button><Plus className="mr-2 h-4 w-4" /> Add Contact</Button>
```

### Cards

```tsx
<Card>
  <CardHeader>
    <CardTitle>Organizations</CardTitle>
    <CardDescription>Manage partner relationships</CardDescription>
  </CardHeader>
  <CardContent>
    {/* Content */}
  </CardContent>
  <CardFooter>
    <Button>View All</Button>
  </CardFooter>
</Card>
```

### Data Tables

```tsx
<DataTable
  columns={columns}
  data={contacts}
  filterableColumns={[
    { id: "contact_type", title: "Type" },
    { id: "status", title: "Status" },
  ]}
  searchableColumns={[
    { id: "name", title: "Name" },
    { id: "email", title: "Email" },
  ]}
/>
```

### Status Badges

```tsx
// Active
<Badge variant="success">Active</Badge>

// Pending
<Badge variant="warning">Pending</Badge>

// Inactive
<Badge variant="secondary">Inactive</Badge>

// Priority
<Badge variant="destructive">Urgent</Badge>
```

## 5. Responsive Behavior

### Breakpoints

```css
/* Mobile First */
sm: 640px   /* Tablet */
md: 768px   /* Small laptop */
lg: 1024px  /* Desktop */
xl: 1280px  /* Large desktop */
2xl: 1536px /* Extra large */
```

### Mobile Adaptations

1. **Sidebar**: Collapses to hamburger menu
2. **Tables**: Horizontal scroll or card view
3. **Kanban**: Single column with tabs
4. **Forms**: Full-width inputs
5. **Dashboard**: Stacked cards

## 6. Accessibility

### WCAG 2.1 AA Compliance

- Color contrast: 4.5:1 minimum
- Focus indicators: Visible on all interactive elements
- Keyboard navigation: Full support
- Screen readers: Proper ARIA labels
- Motion: Respects prefers-reduced-motion

### Focus States

```css
:focus-visible {
  outline: 2px solid var(--blkout-gold);
  outline-offset: 2px;
}
```
